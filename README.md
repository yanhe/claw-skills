## 快速开始

```bash
# 安装 skill
clawhub install <skill-name>

# 从本地安装
clawhub install ./skills/<skill-name>
```

## 开发指南

1. 使用 `clawhub create` 创建新 skill
2. 编辑 `SKILL.md` 定义功能
3. 测试通过后 `clawhub publish`
